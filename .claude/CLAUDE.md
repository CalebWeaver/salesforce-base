# Aimpoint Salesforce Baseline

Generates Salesforce SFDX projects. Run `scripts/init-project.sh` to initialize a project.

Use skills for detailed guidance:
- `/baseline-structure` — project layout, profiles, manifests, skills/agents organization, gotchas
- `/editing-guide` — changing standards, adding profiles, updating skills and templates
- `/init-script` — how the init script works, testing all three profiles
- `/agent-creator` — designing and creating Claude Code agents: frontmatter fields, placement, manifest wiring, prompt patterns
- `/plugins-and-marketplaces` — discovering, installing, and managing plugins; creating public, team, and private marketplaces
